Software & Development Tools

PhotonFlow SDK

Comprehensive development kit for building and deploying optical neural networks with familiar Python APIs. 

import photonflow as pf

model = pf.OpticalNN()
model.add(pf.Linear(784, 256))
model.add(pf.OpticalReLU())
model.add(pf.Linear(256, 10))

model.compile(device='ONN-Edge')
model.fit(X_train, y_train)
