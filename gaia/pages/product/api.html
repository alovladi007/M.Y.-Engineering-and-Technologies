Code Examples

Get started quickly with our comprehensive code examples

Python SDK

import gaia

# Initialize client
client = gaia.Client(
    api_key="your-api-key",
    endpoint="https://api.gaia.energy"
)

# Get cluster metrics
metrics = client.get_metrics(
    cluster_id="prod-cluster-1",
    time_range="1h"
)

# Create optimization policy
policy = client.create_policy({
    "name": "energy-optimization",
    "target_joules_per_request": 150,
    "power_cap_percent": 80
})

# Apply policy
client.apply_policy(
    cluster_id="prod-cluster-1",
    policy_id=policy.id
)

cURL Examples

Get Metrics

curl -X GET \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  "https://api.gaia.energy/v1/metrics?cluster_id=prod-cluster-1"

Create Policy

curl -X POST \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"name": "optimization", "target_joules_per_request": 150}' \
  "https://api.gaia.energy/v1/policies"

API Endpoints

Complete reference for all available API endpoints

GET /v1/metrics
Get cluster energy metrics

Retrieve real-time energy consumption data, power usage, and efficiency metrics for specified clusters.

Rate Limit: 1000/min Authentication: Required
POST /v1/policies
Create energy optimization policy

Create new energy optimization policies with custom parameters and targets.

Rate Limit: 100/min Authentication: Required
PUT /v1/optimize
Trigger optimization process

Manually trigger energy optimization processes for specified clusters or workloads.

Rate Limit: 50/min Authentication: Required
GET /v1/reports
Generate energy reports

Generate comprehensive energy consumption reports with baseline vs treatment analysis.

Rate Limit: 10/min Authentication: Required
