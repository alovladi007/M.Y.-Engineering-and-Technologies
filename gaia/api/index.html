https://api.gaia.energy/v1

Authentication

All API requests require authentication using your API key:

Authorization: Bearer YOUR_API_KEY

Pro Tier 10,000 requests/hour
Enterprise Unlimited

Quick Example

Get Energy Metrics

curl -X GET "https://api.gaia.energy/v1/metrics/energy" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
Response
{
  "status": "success",
  "data": {
    "current_power": 1247,
    "efficiency": 94.2,
    "carbon_saved": 2.1,
    "cost_savings": 47200,
    "timestamp": "2024-01-20T18:00:00Z"
  }
}
GET /v1/metrics/energy
Real-time energy data

Retrieve current energy consumption metrics and optimization status.

Parameters
cluster_id optional
timeframe optional
Response Fields
• current_power (kW)
• efficiency (%)
• carbon_saved (tCO₂)
• cost_savings ($)
POST /v1/optimize
Trigger optimization

Start energy optimization for specified clusters or workloads.

Request Body
{
  "cluster_id": "prod-cluster-1",
  "optimization_level": "aggressive",
  "constraints": {
    "max_power": 1500,
    "carbon_budget": 2.0
  }
}
Response
{
  "status": "success",
  "optimization_id": "opt_123456",
  "estimated_savings": {
    "power": 15,
    "carbon": 0.3,
    "cost": 1200
  }
}
GET /v1/analytics/carbon
Carbon footprint data

Get detailed carbon footprint analysis and reduction metrics.

Query Parameters
start_date required
end_date required
granularity hour|day|week
Response Fields
• total_emissions (tCO₂)
• emissions_saved (tCO₂)
• reduction_percentage (%)
• renewable_energy_usage (%)

Authentication

API Key Authentication

All API requests must include your API key in the Authorization header: 

Authorization: Bearer YOUR_API_KEY

Getting Your API Key

1

Sign up for GAIA

Create your account at gaia.energy

2

Navigate to API Settings

Go to Settings → API Keys in your dashboard

3

Generate API Key

Click "Generate New Key" and copy the key

Security Best Practices

Keep your API key secure

Never expose it in client-side code or public repositories

Rotate keys regularly

Generate new keys periodically and revoke old ones

Use environment variables

Store API keys in secure environment variables

Code Examples

Python

import requests

# Set up the API client
api_key = "YOUR_API_KEY"
base_url = "https://api.gaia.energy/v1"
headers = {
    "Authorization": f"Bearer {api_key}",
    "Content-Type": "application/json"
}

# Get energy metrics
response = requests.get(f"{base_url}/metrics/energy", headers=headers)
data = response.json()

print(f"Current Power: {data['data']['current_power']} kW")
print(f"Efficiency: {data['data']['efficiency']}%")

# Start optimization
optimization_data = {
    "cluster_id": "prod-cluster-1",
    "optimization_level": "aggressive"
}

response = requests.post(
    f"{base_url}/optimize", 
    headers=headers, 
    json=optimization_data
)
result = response.json()
print(f"Optimization ID: {result['optimization_id']}")

JavaScript

const API_KEY = 'YOUR_API_KEY';
const BASE_URL = 'https://api.gaia.energy/v1';

// Create API client
const apiClient = {
  async request(endpoint, options = {}) {
    const response = await fetch(`${BASE_URL}${endpoint}`, {
      ...options,
      headers: {
        'Authorization': `Bearer ${API_KEY}`,
        'Content-Type': 'application/json',
        ...options.headers
      }
    });
    return response.json();
  }
};

// Get energy metrics
async function getEnergyMetrics() {
  const data = await apiClient.request('/metrics/energy');
  console.log(`Current Power: ${data.data.current_power} kW`);
  console.log(`Efficiency: ${data.data.efficiency}%`);
}

// Start optimization
async function startOptimization() {
  const result = await apiClient.request('/optimize', {
    method: 'POST',
    body: JSON.stringify({
      cluster_id: 'prod-cluster-1',
      optimization_level: 'aggressive'
    })
  });
  console.log(`Optimization ID: ${result.optimization_id}`);
}
