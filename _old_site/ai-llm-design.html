System Architecture

Literature → Knowledge Graph

  • Ingest: PDFs → XML with GROBID
  • Extract: ChemDataExtractor + MatSciBERT
  • Normalize: OPSIN/SMILES/InChI
  • Store: Neo4j + parquet tables

LLM Layer (RAG)

  • Retriever: BM25 + dense embeddings
  • Planner: Tool orchestration
  • Prompting: Function-calling style
  • Output: YAML contracts

Predictors

  • Structure: ALIGNN/CGCNN/MEGNet
  • Spectra: XRD/Raman/TEM models
  • Text-only: LLM embeddings
  • Morphology: 81% accuracy

Inverse Design

  • Search: Bayesian Optimization
  • Generation: Diffusion models
  • Safety: eNanoMapper checks
  • Autonomy: Human-in-the-loop

Example Workflows

A) Text-Only → Predicted Morphology

User Input:

"Seed-mediated Au in CTAB + Ag+ co-ion; mild AA; 25–30 °C."

System Process:

  • • Embed description using MatSciBERT
  • • Classifier predicts rod vs sphere vs cube
  • • Estimates aspect ratio (AR≈3–4)
  • • Provides confidence score (81% benchmark)

LLM Output:

"Predicted rod, AR≈3–4; suggest UV-Vis check (LSPR 760–820 nm) and TEM segmentation pipeline."

B) Inverse Design (Plasmonic Target)

Goal:

800 nm LSPR in water with Au nanorods

Planner (LLM):

  • • Proposes seed recipe + CTAB/AgNO₃/AA ranges
  • • Notes AR↔LSPR trend
  • • Outputs 12-run DOE

Optimization:

  • • Gans-surrogate + image-in-the-loop
  • • BO minimizes |LSPR–800|
  • • Constraints on toxicity/surfactant

Report:

Best conditions: T=28°C, [Ag+] 0.08–0.12 mM, AA 0.6–0.7 mM with uncertainty bands
