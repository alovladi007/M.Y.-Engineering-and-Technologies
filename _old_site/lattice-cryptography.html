Mathematical Foundation

The mathematical principles behind lattice-based cryptography

Learning With Errors (LWE)

The Learning With Errors problem is the mathematical foundation of lattice-based cryptography. It involves finding a secret vector given noisy linear equations.

LWE Problem Definition:

Given $(A, b = As + e)$ where:

  • ‚Ä¢ $A \in \mathbb{Z}_q^{n \times m}$ is a random matrix
  • ‚Ä¢ $s \in \mathbb{Z}_q^n$ is the secret vector
  • ‚Ä¢ $e \in \mathbb{Z}_q^m$ is a small error vector
  • ‚Ä¢ Find $s$ given $(A, b)$

Security Guarantee:

The LWE problem is believed to be hard even for quantum computers, making it suitable for post-quantum cryptography.

Lattice Structure

A lattice is a discrete additive subgroup of $\mathbb{R}^n$ generated by linearly independent vectors $b_1, b_2, \ldots, b_k$.

Lattice Definition:

$\mathcal{L}(B) = \{ \sum_{i=1}^k x_i b_i : x_i \in \mathbb{Z} \}$

where $B = [b_1, b_2, \ldots, b_k]$ is the basis matrix

Key Properties:

  • ‚Ä¢ Discrete structure resists quantum attacks
  • ‚Ä¢ Hard problems: SVP, CVP, SIS
  • ‚Ä¢ Efficient arithmetic operations
  • ‚Ä¢ Small key and ciphertext sizes

Code Example

Kyber KEM implementation in Python:

import numpy as np
from kyber import Kyber768

# Initialize Kyber-768 (NIST Level 3)
kyber = Kyber768()

# Generate key pair
public_key, secret_key = kyber.keygen()

# Encapsulate (encrypt) a shared secret
ciphertext, shared_secret = kyber.encaps(public_key)

# Decapsulate (decrypt) the shared secret
decrypted_secret = kyber.decaps(ciphertext, secret_key)

# Verify correctness
assert shared_secret == decrypted_secret
print("‚úì Kyber KEM successful")
