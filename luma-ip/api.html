Patent Drafting

Generate patent applications using AI

POST /api/v1/patents/draft

curl -X POST https://api.lumaip.com/v1/patents/draft \ -H "Authorization: Bearer YOUR_API_KEY" \ -H "Content-Type: application/json" \ -d '{ "title": "AI-Powered Medical Diagnosis System", "description": "A system for automated medical diagnosis...", "inventors": ["John Doe", "Jane Smith"], "type": "utility" }'

Response

{ "id": "pat_123456", "status": "completed", "claims": [ "A medical diagnosis system..." ], "specification": "FIELD OF THE INVENTION...", "created_at": "2024-01-15T10:30:00Z" }

Prior Art Search

Search global patent databases

GET /api/v1/patents/search

curl -X GET "https://api.lumaip.com/v1/patents/search?q=AI+medical+diagnosis&limit=10" \ -H "Authorization: Bearer YOUR_API_KEY"

Response

{ "results": [ { "patent_number": "US10,123,456", "title": "AI Medical Diagnosis System", "relevance_score": 0.92, "filing_date": "2020-03-15" } ], "total": 247 }
