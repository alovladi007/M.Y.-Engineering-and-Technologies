AI/LLM-Assisted Design Platform

Advanced materials discovery and device optimization powered by state-of-the-art language models and machine learning

System Architecture

Literature → Knowledge Graph

  • Ingest: PDFs → XML with GROBID
  • Extract: ChemDataExtractor + MatSciBERT
  • Normalize: OPSIN/SMILES/InChI
  • Store: Neo4j + parquet tables

LLM Reasoning + RAG

  • Retriever: BM25 + dense embeddings
  • Planner: Tool orchestration
  • Prompting: Function-calling style
  • Output: YAML contracts

Specialist Predictors

  • Structure: ALIGNN/CGCNN/MEGNet
  • Spectra: XRD, Raman, TEM/SEM
  • Text-to-property: LLM embeddings
  • Validation: 81% accuracy (ITMO)

Inverse Design

  • Optimization: Bayesian/RL/Tree-search
  • Generative: Diffusion/transformers
  • Safety: Toxicity checks
  • Autonomy: Human-in-the-loop

  • • ChemDataExtractor for chemical entities

Structured Prediction

  • • Fine-tuning on JARVIS/MP

Text-Only Prediction

Early stage ideas without structure:

  • • 81% accuracy benchmark

Multimodal Analysis

3. Hypothesis Generation & Experimental Design

LLM-Driven Design of Experiments

Input:

  • • Target properties and constraints
  • • Available reagents and equipment
  • • Safety and cost limitations

Output:

  • • Candidate synthesis recipes
  • • DOE matrix with factor ranges
  • • Justifications and safety notes

Optimizer: Bayesian optimization loop with Gaussian Process/TPE for parameter selection

4. Inverse Design

Structural Targets

Generative models propose morphologies/compositions → Physics filters (e.g., Mie/Gans for plasmonics) → GNN property verification → LLM synthesis route generation

Process Optimization

Tree search over synthesis sequences with constraints → Bayesian optimization for continuous parameters → Safety and feasibility checks

    type: "nanorod"

  ligands: ["CTAB", "Ag+"]

    - name: "LSPR_longitudinal_nm"
      value: 800

synthesis_step.yaml

step:
  method: "seed-mediated growth"
  reagents:
    - name: "HAuCl4", conc_mM: 0.5
    - name: "CTAB", conc_mM: 100
    - name: "AgNO3", conc_mM: 0.1
  conditions: 
    T_C: 28
    pH: 2.5
    time_min: 30
  characterization: ["UV-Vis", "TEM"]
  outcomes: 
    morphology: "rod"
    LSPR_nm: 780-820

Example Workflows

A) Text-Only → Predicted Morphology

User Input:

"Seed-mediated Au in CTAB + Ag+ co-ion; mild AA; 25–30 °C"

System Process:

  • 1. Embed description using MatSciBERT
  • 2. Classifier predicts morphology type
  • 3. Estimate aspect ratio and confidence

Output:

"Predicted rod, AR≈3–4 (81% confidence); suggest UV-Vis check (LSPR 760–820 nm) and TEM verification"

B) Inverse Design (Plasmonic Target)

Goal:

800 nm LSPR in water with Au nanorods

Planner Output:

  • • Seed recipe with CTAB/AgNO₃/AA ranges
  • • 12-run DOE matrix
  • • AR↔LSPR trend analysis

Optimization Result:

T=28°C, [Ag+] 0.08–0.12 mM, AA 0.6–0.7 mM with uncertainty bands and literature evidence
