System Architecture

Literature → Knowledge Graph (KG)

Ingestion Pipeline

  • PDFs → XML: GROBID for structured extraction Parse text, tables, references automatically
  • Chemical Extraction: ChemDataExtractor (CDE) Domain-specific NER for materials and properties
  • NLP Models: MatSciBERT/MatBERT Fine-tuned transformers for materials science

Data Normalization & Storage

  • Chemical Normalization: OPSIN/SMILES/InChI Standardized molecular representations
  • Ontologies: eNanoMapper, NanoMine schema Structured vocabularies for consistency
  • Storage: Neo4j graph + Parquet tables Flexible querying and analytics

LLM Layer (Reasoning + RAG)

Retriever

BM25 + dense embeddings (MatSciBERT) for hybrid search over KG and documents

Planner

Instruction-tuned LLM orchestrating tools: literature_scout, extract_recipe, predict_property

Prompting

Tool-former style function-calling with YAML I/O contracts for structured data

Specialized Predictors

Structure/Property

ALIGNN, CGCNN, MEGNet for band gap, stability, elastic moduli

XRD Analysis

CNN/Transformers for phase identification from patterns

Raman Spectroscopy

RamanFormer for mixture quantification and denoising

Microscopy

U-Net for TEM/SEM morphology segmentation

step:
  method: "seed-mediated growth"
  reagents:
    - {name: "HAuCl4", conc_mM: 0.5}
    - {name: "CTAB", conc_mM: 100}
    - {name: "AgNO3", conc_mM: 0.1}
    - {name: "ascorbic acid", conc_mM: 0.64}
  conditions: 
    T_C: 28
    pH: 2.5
    order: ["CTAB","HAuCl4","AgNO3","AA","seed"]
    time_min: 30
  characterization: ["UV-Vis", "TEM"]
  outcomes: 
    morphology: "rod"
    AR: 3.5-3.8
    LSPR_nm: 780-820

Example Workflows

Text → Morphology Prediction

User Input:

"Seed-mediated Au in CTAB + Ag+ co-ion; mild AA; 25–30 °C."

System Process:

  • • Embed description using MatSciBERT
  • • Classify morphology: rod vs sphere vs cube
  • • Estimate aspect ratio with confidence

Output:

"Predicted: Rod, AR≈3–4 (81% confidence). Suggest UV-Vis check (LSPR 760–820 nm) and TEM validation."

Inverse Design for Plasmonics

Target:

800 nm LSPR in water with Au nanorods

Planner Output:

  • • Seed recipe + CTAB/AgNO₃/AA ranges
  • • 12-run DOE for optimization
  • • AR↔LSPR trend analysis

Optimization Result:

T=28°C, [Ag+] 0.08–0.12 mM, AA 0.6–0.7 mM with uncertainty bands + literature evidence
